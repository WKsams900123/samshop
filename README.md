# 山姆优选代购 - PWA 版

## 这是什么

一个**可以装到手机桌面**的代购下单页面。客户打开后扫码或点链接选购商品、提交订单，你从手机后台管理订单。

- 🆓 **完全免费** - 托管在 GitHub Pages，不需要服务器
- 📱 **像APP一样** - 支持 PWA，可添加到手机桌面，全屏体验
- 🔒 **隐私安全** - 不采集个人信息，每次下单有隐私提示
- ☁️ **可选云端** - 配好 Supabase 后多设备同步，不配也能用本地版

---

## 快速开始（3步，无需编程）

### 第1步：注册 GitHub

去 [github.com](https://github.com) 免费注册一个账号。

### 第2步：上传文件

1. 登录 GitHub
2. 点击右上角 **+** → **New repository**
3. 仓库名填 `samshop`，选 **Public**，点击 **Create repository**
4. 点击 **uploading an existing file**
5. 把 `index.html`、`manifest.json`、`sw.js` 三个文件拖进去
6. 点击 **Commit changes**

### 第3步：开启 Pages

1. 进入仓库 → **Settings** → 左侧 **Pages**
2. **Branch** 选 `main`，点击 **Save**
3. 等1-2分钟，页面顶部会显示 `Your site is live at https://你的用户名.github.io/samshop/`

✅ 完成！打开这个地址就能用了。

## 进阶：配置云端数据库（可选）

默认情况下，订单存在浏览器本地（不同设备看不到对方的订单）。

如果你想**多设备同步**（电脑、手机都能看订单），可以配置 Supabase：

### 1. 注册 Supabase（免费）

去 [supabase.com](https://supabase.com) 注册，创建新项目。

### 2. 建表

在 Supabase 项目左侧菜单 → **SQL Editor** → 粘贴 `setup.sql` 的内容 → 点 **Run**。

### 3. 获取配置

左侧菜单 → **Settings** → **API**：
- 复制 **Project URL**（类似 `https://xxxxx.supabase.co`）
- 复制 **anon public** key（类似 `eyJh...`）

### 4. 填入系统

打开你的代购页面 → 点右下角 ⚙ → 输入密码进入后台 → **设置** 标签 → 粘贴上面两个值 → **保存设置**。

## 客户使用流程

1. 你发二维码或链接给客户
2. 客户打开 → 看到商品列表 → 选商品加购物车
3. 点「去结算」→ 看到**隐私安全提示** → 填写姓名手机号 → 勾选同意 → 提交
4. 下单成功 → 可复制订单信息发给你
5. **建议客户点底部提示「添加到桌面」**，下次从桌面图标一键进入

## 你管理订单流程

1. 打开页面 → 点右下角 ⚙
2. 输入管理密码 → 进入后台
3. 看到所有订单 → 点「确认接单」→ 处理 → 点「标记完成」
4. 统计标签页可看总订单数、营收等数据
